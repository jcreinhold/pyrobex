Python bindings for ROBEX brain extraction.

This package comes with ROBEX v1.2 for Linux. Windows is not currently supported.

* Free software: BSD license
* Documentation: https://pyrobex.readthedocs.io.


Install
-------

.. code-block:: bash
    pip install pyrobex
